# Table 2: Correlations with Statistical Significance (p-values)
ca_corrs = corr_matrix['CA1'].drop('CA1').sort_values(ascending=False)

# Calculate p-values for each correlation
p_values = []
for var in ca_corrs.index:
    r, p = stats.pearsonr(df['CA1'].dropna(), df.loc[df['CA1'].notna(), var])
    p_values.append(p)

corr_table = pd.DataFrame({
    'Variable': ca_corrs.index,
    'Correlation (r)': ca_corrs.values.round(3),
    'p-value': [f'{p:.4f}' for p in p_values],
    'Significant (p<0.05)': ['Yes' if p < 0.05 else 'No' for p in p_values],
    'Interpretation': ['Strong' if abs(r) > 0.5 else 'Moderate' if abs(r) > 0.3 
                       else 'Weak' for r in ca_corrs.values]
})
print("Table 2: CA1 Correlations with Outcome Variables")

# Table 3: High vs Low CA Group Comparison with T-Tests
ca_median = df['CA1'].median()
df['CA1_binary'] = df['CA1'].apply(lambda x: 'High CA' if x >= ca_median else 'Low CA')

high_ca = df[df['CA1_binary'] == 'High CA']
low_ca = df[df['CA1_binary'] == 'Low CA']

# Calculate t-tests for each outcome variable
test_results = []
for var in ['GO1', 'NPS1', 'SE1', 'RLS1']:
    t_stat, p_val = stats.ttest_ind(high_ca[var].dropna(), low_ca[var].dropna())
    test_results.append({
        'Outcome': var,
        'High CA Mean': high_ca[var].mean().round(2),
        'Low CA Mean': low_ca[var].mean().round(2),
        'Difference': (high_ca[var].mean() - low_ca[var].mean()).round(2),
        't-statistic': round(t_stat, 2),
        'p-value': f'{p_val:.4f}',
        'Significant': 'Yes' if p_val < 0.05 else 'No'
    })

comparison_stats = pd.DataFrame(test_results)
print(f"Table 3: High CA ({ca_median:.0f}, n={len(high_ca)}) vs Low CA (<{ca_median:.0f}, n={len(low_ca)})")
comparison_stats

Summary of Key Findings

Key observations from Conflict Adaptation (CA1) analysis:

Distribution: Most teams report moderate-to-high conflict adaptation scores (median around 8-9 out of 10)

Correlations: CA1 shows positive correlations with all outcome variables. Teams that handle conflict well tend to report better outcomes.

Strongest relationships: CA1 correlates most strongly with RLS1 (relationship strength) and NPS1 (team satisfaction). Constructive conflict handling appears linked to stronger interpersonal bonds.

High vs Low CA: Teams with above-median CA1 scores show consistently higher means across all outcome metrics.

Section differences: Some variation exists between sections in conflict adaptation patterns.
